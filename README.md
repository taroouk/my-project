1. Core Infrastructure & UI :
• Public-facing Servly Main Website (showcasing Servly, Career Opportunities, Partners).  
• Login gateways for Customers and Merchants to access their Dashboards.  
• Displaying Packages / Plans with a booking feature for Meeting / Demo Session.  
2. Website Builder System :
• Ability for merchants to create their own websites using a customizable Subdomain or Custom Domain.  
• Multiple theme options and the "Build Your Own Website" feature.  
• Integration to add Services, Products, and Subscriptions to the merchant's site.  
3. Booking and Customer Management (CRM) :
• Merchant control over Service Duration and Buffer Time / Gap Time.  
• Setting Booking Capacity / Service Slots (e.g., 2 Online slots).  
• Merchant control over Opening Hours and Closing Hours.  
• Customer journey: selecting Service, Upsell/Add-ons (Frequently Purchased Together), Complementary Services/Offers, choosing date/time on the Booking Calendar, and paying an Online Deposit.  
• System for controlling WhatsApp messages (Triggers and timing).  
4. Loyalty Systems and Reporting :
• Loyalty Programs management (Free Service, Cash Discount, Points System).  
• Tier-Based System setup (e.g., Silver Tier for 5% discount based on spending).  
• Dynamic Loyalty Card display (showing progress, e.g., 3 washes done, 1 left).  
• Sales, Consumables, Expenses, and Net Profit reports.  
• Customer Reports (New Customers Count, Returning Customer Rate, Highest Paying Customer).  
• Customer Portal showing Loyalty Points Overview.  
5. HR and Servly Admin Panel :
• Merchant registration of employee details (Full Name, Salary, Direct Manager, Annual Leave Balance, etc.).  
• Attendance using Location-based Attendance.  
• Managing Leave Requests.  
• Attendance Reports (Employee Attendance Score, Most/Least Committed Employee).  
• Servly Admin Panel to manage Merchants Accounts and Platform Settings.  
• Implementation of the Permissions System for various staff roles
